## When to use embed

Use embed to insert an inline frame. An inline frame or `iFrame` is used to embed content in a webpage.

## How to use embed

### Accessibility

`iFrame`'s need to be titled using the `title` attribute. The title should identify to assistive technologies what the embeded content is.
